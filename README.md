#### python3-alt:

snap python3 packaging of:

- `python3.12`
- `python3.11`
- `python3.10`
- `python3.9`
- `python3.8`

run:
```bash
# for python3.12 
python3-alt.3-12 --version
# for python3.11
python3-alt.3-11 --version
# for python3.10
python3-alt.3-10 --version
# for python3.9 
python3-alt.3-9 --version
# for python3.8 
python3-alt.3-8 --version
```
